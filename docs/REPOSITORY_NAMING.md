# Repository Naming

Course repository names are derived from the course folder name.

Rules:

- lowercase the repo name
- replace spaces and punctuation with hyphens
- preserve the course code at the beginning
- avoid duplicate repo names by adding a short suffix only when needed

Examples:

- `M_1AlL Linear algebra` -> `m-1all-linear-algebra`
- `MF2CAF Special chapters on functional analysis` -> `mf2caf-special-chapters-on-functional-analysis`
- `A4RNe Neural networks` -> `a4rne-neural-networks`

# curricula.live Import Notes

Each course repository is a semantic course package.

A future importer should read:

- `data/course.json`
- `data/concept.json`
- `data/relation.json`
- `data/curricula.json`
- `data/material.json`
- `data/source.json`
- `data/curriculum_plan.json` when present

The importer should preserve:

- course identity
- material links
- concept ids
- relation ids
- confidence
- evidence
- manual-review status

This is not yet an official syllabus graph. It is a seed dataset for future review.

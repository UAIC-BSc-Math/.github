
# UAIC-BSc-Math organization documentation

This repository contains organization-level documentation, the organization profile README, default issue templates, a pull request template, and workflow templates for the UAIC-BSc-Math archive.

Course repositories are separate and are not stored here.

## Layout

- `profile/README.md` - organization profile README
- `docs/` - publishing status, repository index, metadata status, and maintenance notes
- `ISSUE_TEMPLATE/` - issue forms for metadata and material review
- `workflow-templates/` - template workflows for validation and markdown checks

No course PDFs live in this repository.

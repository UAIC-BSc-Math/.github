
# UAIC-BSc-Math

A private course-archive and semantic-metadata organization for UAIC mathematics study materials.

## Overview

- 95 private course repositories have been published.
- The repositories are course-centered and mirror the cleaned study archive.
- Each repository contains navigation README files, archived materials, semantic metadata, and material/source registries.
- The remaining manual-review repositories are intentionally excluded from broad publishing.

## Metadata model

Each course repository uses a seed semantic layer built from machine-readable JSON files:

- `data/course.json`
- `data/concept.json`
- `data/relation.json`
- `data/curricula.json`
- `data/year.json`
- `data/unit.json`
- `data/lesson.json`
- `data/homework.json`
- `data/seminary.json`
- `data/material.json`
- `data/source.json`
- `data/curriculum_plan.json` where present

## Status

## Private / rights note

These repositories are private because the archive may contain course materials, professor-provided PDFs, books, and downloaded files whose redistribution rights are unclear. Public release requires separate review.
